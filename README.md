# Web query expansion

## Quick start

First, you will need to clone the repo.
```
git clone https://github.com/alannagenin/web-query.git
cd web-query
```

Then, we will set a virtual environment.
```python
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
